## рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

### ­ЪЊї рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ

**Softgen Рђћ рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ (Admin/User рЃарЃЮрЃџрЃћрЃЉрЃўрЃЌ, JWT, Projects/Banners, A/B рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ, Scheduler Engine)**

### ­Ъј» рЃЏрЃўрЃќрЃљрЃюрЃў

рЃЋрЃћрЃЉрЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА:
- РюЁ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљ рЃЏрЃљрЃарЃЌрЃЋрЃљрЃАрЃљ рЃЊрЃљ рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљрЃА (Admin Рєњ Invite, User Рєњ Activation)
- РюЁ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃА JWT-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃўрЃЌ
- РюЁ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџ CRUD-рЃА
- РюЁ A/B рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљрЃА CTR/Conversion-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
- РюЁ рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљрЃАрЃљ рЃЊрЃљ рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃБрЃџ рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљрЃА (Scheduler Engine)
- РюЁ рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃљрЃдрЃЊрЃњрЃћрЃюрЃўрЃА рЃАрЃарЃБрЃџ рЃюрЃљрЃЎрЃљрЃЊрЃА

рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃўрЃљ рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃюрЃњрЃБрЃџрЃў рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА, рЃЊрЃўрЃАрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.

### ­ЪЊЈ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃў

| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў | рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў |
| --- | --- |

| ­ЪќЦ№ИЈ рЃЊрЃћрЃърЃџрЃЮрЃўрЃЏрЃћрЃюрЃбрЃў | рЃерЃўрЃЊрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃљрЃю Cloud (Docker/K8s-ready) |
| ­ЪЊ▒ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ | Web Application (Desktop & Mobile Responsive) |

## РџЎ№ИЈ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў

### ­Ъћљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃћрЃЉрЃў

#### ­ЪћЉ Admin рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ (рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў)
- POST `/api/v1/admin/auth/signup` Рђћ рЃЌрЃБ рЃљрЃЊрЃЏрЃўрЃюрЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА Рєњ 409
- Email рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃўрЃљ; рЃърЃљрЃарЃЮрЃџрЃў рЃФрЃџрЃўрЃћрЃарЃў (min 8, рЃарЃЌрЃБрЃџрЃў regex)

#### ­ЪЉц User рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ
- POST `/api/v1/admin/users` Рђћ Admin рЃЦрЃЏрЃюрЃўрЃА, рЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ `activationToken`
- POST `/api/v1/auth/activate` Рђћ User рЃљрЃДрЃћрЃюрЃћрЃЉрЃА рЃърЃљрЃарЃЮрЃџрЃА рЃбрЃЮрЃЎрЃћрЃюрЃўрЃЌ (single-use, expirable)

#### ­ЪћЊ Login/Logout/Me
- POST `/api/v1/auth/login` Рђћ JWT рЃбрЃЮрЃЎрЃћрЃюрЃў + `expires_in`
- GET `/api/v1/users/me` Рђћ рЃърЃарЃЮрЃцрЃўрЃџрЃў JWT-рЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ
- POST `/api/v1/auth/logout` Рђћ рЃбрЃЮрЃЎрЃћрЃюрЃўрЃА рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃљ (blacklist рЃЌрЃБ рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ)

#### ­Ъћё Password Reset
- POST `/api/v1/auth/forgot-password` Рђћ resetToken рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ email-рЃќрЃћ
- POST `/api/v1/auth/reset-password` Рђћ рЃљрЃ«рЃљрЃџрЃў рЃърЃљрЃарЃЮрЃџрЃў resetToken-рЃўрЃЌ (single-use, expirable)

### ­ЪЊЂ User Panel Рђћ Projects CRUD
- GET `/api/v1/projects` Рђћ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ + рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ/рЃърЃљрЃњрЃўрЃюрЃљрЃфрЃўрЃљ
- POST `/api/v1/projects` Рђћ рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
- GET `/api/v1/projects/{projectId}` Рђћ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ (ownership check)
- PUT/PATCH `/api/v1/projects/{projectId}` Рђћ рЃАрЃарЃБрЃџрЃў/рЃюрЃљрЃгрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
- DELETE `/api/v1/projects/{projectId}` Рђћ Soft delete рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ

### ­Ъќ╝№ИЈ User Panel Рђћ Banners CRUD
- GET `/api/v1/projects/{projectId}/banners` Рђћ рЃАрЃўрЃљ
- POST `/api/v1/projects/{projectId}/banners` Рђћ рЃерЃћрЃЦрЃЏрЃюрЃљ (title, imageUrl, targetUrl, notes)
- GET `/api/v1/projects/{projectId}/banners/{bannerId}` Рђћ рЃюрЃљрЃ«рЃЋрЃљ
- PUT/PATCH `/api/v1/projects/{projectId}/banners/{bannerId}` Рђћ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
- DELETE `/api/v1/projects/{projectId}/banners/{bannerId}` Рђћ рЃгрЃљрЃерЃџрЃљ/рЃљрЃарЃЦрЃўрЃЋрЃљрЃфрЃўрЃљ

### ­ЪДЉРђЇ­Ъњ╝ Admin Panel
- Users: GET, GET/{id}, PATCH/{id}, DELETE/{id}
- Projects: GET, GET/{id}, PATCH/{id}, DELETE/{id}
- Banners: GET, GET/{id}, PATCH/{id}, DELETE/{id}

### ­ЪДф A/B рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ
- рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃў: `variant_group, impressions, clicks, conversions, ctr, conversion_rate`
- CTR/Conversion Rate рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ `ABTestingService`-рЃерЃў
- рЃбрЃћрЃАрЃбрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃАрЃљрЃА Рђћ рЃЊрЃљрЃЉрЃљрЃџрЃў CTR/CR рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃў рЃърЃљрЃБрЃќрЃљрЃерЃў, рЃњрЃљрЃЏрЃљрЃарЃ»рЃЋрЃћрЃЉрЃБрЃџрЃў рЃљрЃЦрЃбрЃўрЃБрЃарЃў
- Report: GET `/api/v1/admin/abtesting/report`

### РЈ▒№ИЈ Scheduler Engine
- рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃў: `start_time, end_time, priority`
- `BannerSchedulerService` рЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃА/рЃљрЃЕрЃћрЃарЃћрЃЉрЃА рЃЊрЃарЃЮрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
- рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃБрЃџрЃў рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ: рЃЏрЃљрЃдрЃљрЃџрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃў Рєњ рЃБрЃцрЃарЃЮ рЃ«рЃерЃўрЃарЃў рЃўрЃЏрЃърЃарЃћрЃАрЃўрЃћрЃЉрЃў
- рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў: GET `/api/v1/admin/scheduler/status`
